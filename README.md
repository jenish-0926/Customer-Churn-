                                                Introduction

Customer churn means customers stop using a service. It causes loss for businesses. This project uses synthetic data to study customer behavior and find reasons for churn using EDA.

                                                  Dataset Overview

The dataset contains 1,00,000 customer records with features like:

Age, Gender
Tenure
Balance
Credit Score
Salary
Number of Products
Active Status
Churn (0 = No, 1 = Yes)

                                                      Methodology
Generate synthetic data
Clean and preprocess data
Convert categorical values
Perform EDA
Use charts like bar, pie, histogram, scatter, and heatmap

                                                 Statistical Analysis
Calculated mean, median, mode, and standard deviation
Compared churn vs non-churn customers
Found:
Low tenure → high churn
Inactive users → more churn
Fewer products → more churn

                                                         Conclusion

Churn mainly depends on customer activity, tenure, and product usage.
Businesses can reduce churn by improving engagement and offering better services.
